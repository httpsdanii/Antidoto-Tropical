# O Antídoto Tropical

## 📌 Descrição

**O Antídoto Tropical** é um jogo desenvolvido no Construct 3 como parte de um trabalho acadêmico da disciplina de Desenvolvimento de Jogos, realizado em dupla.
O trabalho foi desenvolvimento em aproximadamente um mês, seguindo os requisitos definidos na disciplina, incluindo criação de mecânicas, ambientação inspirada na Baixada Santista e implementação completa do jogo utilizando Construct 3.

## 📖 História

A Ilha das Cobras é um local isolado onde apenas pessoas autorizadas podem entrar, devido à presença de serpentes extremamente venenosas.
Entretanto, o verdadeiro motivo da proibição está escondido no subsolo da ilha, onde existe um laboratório secreto que realiza experimentos com mutações em cobras.

Um dos experimentos deu errado, criando uma cobra gigante e extremamente inteligente, que fugiu do laboratório durante a noite e espalhou ovos nos barcos dos cientistas.
No dia seguinte, a cidade de Santos foi infestada por cobras venenosas.

A única pessoa que conhece a existência do laboratório é o prefeito, e sua missão é chegar até a ilha, encontrar o antídoto e salvar a população antes que seja tarde.

## 🎮 Como jogar

* Movimentação: teclas **WASD** ou **setas do teclado**
* O objetivo é chegar até o final do mapa e encontrar o antídoto
* O jogador deve completar o percurso sem cair ou encostar no veneno das cobras espalhado pelo chão
* O jogo possui desafios de plataforma e parkour

### ✔️ Como vencer

Conseguir chegar até o final da fase e coletar o antídoto.

### ❌ Como perder

* Encostar no veneno das cobras
* Errar os saltos durante o percurso

## ▶️ Como baixar e abrir o projeto no Construct 3

1. Clique no botão **Code** neste repositório
2. Clique em **Download ZIP**
3. Extraia o arquivo baixado para uma pasta no seu computador
4. Abra o Construct 3
5. Clique em **Open**
6. Selecione a pasta do projeto ou o arquivo do Construct dentro da pasta
7. Aguarde o carregamento do projeto e execute o jogo pelo editor

Obs: O projeto foi desenvolvido no Construct 3, portanto é necessário utilizar essa versão para abrir e executar corretamente.
